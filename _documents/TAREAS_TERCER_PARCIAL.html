LAN-01 Instalar Netcode for GameObjects y configurar el NetworkManager
Sin Asignar ⏱ 1.5 horas Fácil
▶

Antes de cualquier tarea de red, hay que agregar el package de Unity Netcode for GameObjects (NGO) al proyecto y configurar el NetworkManager como punto central de todas las conexiones LAN.

  1. Abrir Window → Package Manager y buscar "Netcode for GameObjects". Instalarlo.
  2. En la escena Gameplay, crear un GameObject vacío llamado NetworkManager.
  3. Agregarle el componente NetworkManager y como transport seleccionar UnityTransport.
  4. En el componente UnityTransport, verificar que el protocolo sea UDP y el puerto 7777.
  5. Agregar el prefab del jugador al campo Player Prefab del NetworkManager.
  6. Hacer una prueba: con dos instancias del editor (ParrelSync o builds), verificar que una puede hacer Host y la otra conectarse como Client.
